# kommon-mvi

Suggestion for Model View Intent inspired Android applications written in Kotlin

Add it to your build.gradle with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and:

```gradle
dependencies {
    implementation "cc.femto:kommon-mvi:2.1.0"
}
```
